# Memory 一致性

Agent 的“记忆”必须一致，否则行为会混乱

典型问题：

  • 同一事实被多次修改（冲突）
  • 旧记忆覆盖新记忆
  • 不同 Agent 记忆不一致
  • hallucination 写入 memory
Memory = Persistent State

但问题是：

LLM 本身是“无状态”的

所以需要：

外部 memory 系统 + 一致性机制

# 解决思路

1️⃣ Memory 分层

Working Memory（短期）
Long-term Memory（长期）

2️⃣ 写入控制（关键）

  • 不是什么都存

  • 需要：

    • validation

      更偏向存储长期记忆

    • deduplication

      去掉重复的信息

    • scoring

3️⃣ 一致性策略

  • versioning（版本控制）

    同一条知识的多个版本

  • conflict resolution

    Memory 中内容相对的信息是，如何选择？

    • 时间
    • 置信度
    • 多值共存（有时 A，有时 B）

  • source tracking（来源）
